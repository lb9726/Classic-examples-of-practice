vi /etc/network/interface
在window下可以给一个网卡配置多IP的功能，那么ubuntu下可不可以呢？理论上应该是可以的，跟平台应该没有什么大的关系

# This file describes the network interfaces available on your system
# and how to activate them. For more information, see interfaces(5).

source /etc/network/interfaces.d/*

# The loopback network interface
auto lo
iface lo inet loopback

# The primary network interface
#auto eno1
#iface eno1 inet static
#address 192.168.168.166
#netmask 255.255.255.0

#will config dynamic allocation IP 
auto eno1
auto eno1:0
auto eno1:1

iface eno1:0 inet static
address 192.168.168.166
netmask 255.255.255.0

iface eno1:1 inet dhcp


以上将eno1的配置一个静态IP，一个动态IP的，测试了一下，发现可以使用，这还是个不错的功能,只是每次插拔更换网线即可，不需要每次都配置后再插拔网线

service networking restart 命令需要在root下执行，在普通用户下执行不会报错，但是没有作用
